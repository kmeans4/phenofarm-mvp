# Dispensary Marketplace Feature List
## Research-Driven B2B Cannabis Wholesale Features

### VIEW MODES
1. **Grid/List Toggle** - Allow dispensaries to switch between card grid view and detailed list view
2. **Infinite Scroll** - Replace pagination with smooth infinite scrolling for product discovery
3. **Sticky Header Filters** - Filter bar stays visible when scrolling through products

### BROWSE MODES
4. **Browse by Shop/Vendor** - View all products from a specific grower with their branding
5. **Browse All Products** - Cross-vendor product discovery with universal filtering
6. **Recently Added** - Quick filter for new inventory (last 7 days)
7. **Trending/Most Popular** - Products with highest order volume

### FILTERING & SORTING
8. **Price Sort** - Low to high, high to low
9. **THC% Sort/Filter** - Sort by potency, filter by THC range (e.g., 15-20%, 20-25%, 25%+)
10. **CBD% Sort/Filter** - Same pattern for CBD content
11. **Product Type Filter** - Flower, Pre-rolls, Edibles, Concentrates, Vapes, Topicals, etc.
12. **Strain Type Filter** - Indica, Sativa, Hybrid
13. **Grower Location Filter** - In-state vs out-of-state
14. **Availability Filter** - In-stock only vs show all
15. **Minimum Order Filter** - Filter by MOQ requirements

### PRODUCT CARDS (Grid View)
16. **Product Image** - High quality with zoom on hover
17. **Strain Name** - Prominent display
18. **Grower Name** - With link to their shop page
19. **THC/CBD % badges** - Quick potency indicators
20. **Price per unit** - Clear pricing with unit (g, oz, lb)
21. **MOQ badge** - Minimum order quantity indicator
22. **Stock status** - "In Stock" / "Low Stock" / "Out of Stock"
23. **Quick Add Button** - One-click add to cart from grid
24. **Test Results Link** - Lab results accessible without leaving page

### PRODUCT LIST (List View)
25. **Compact row layout** - More products visible at once
26. **Full description preview** - 2-line description snippet
27. **Compare checkbox** - Select multiple for side-by-side comparison
28. **Batch/lot info** - For compliance tracking
29. **Last updated** - When inventory was refreshed

### VENDOR SHOP PAGES
30. **Grower banner/header** - Branded shop page with grower logo/banner
31. **Grower bio/about** - Story, cultivation methods, certifications
32. **Grower stats** - Total products, orders fulfilled, rating
33. **Contact grower** - Direct message button
34. **Follow grower** - Get notified of new products
35. **Grower's full catalog** - All their products with same filtering

### SEARCH & DISCOVERY
36. **Universal search** - Search by strain name, grower, or product type
37. **Search suggestions** - Autocomplete with popular strains
38. **Recent searches** - Quick access to past queries
39. **Filter chips** - Visual tags showing active filters with remove option
40. **Clear all filters** - One-click reset

### MOBILE OPTIMIZATIONS
41. **Bottom sheet filters** - Slide-up filter panel on mobile
42. **Swipeable cards** - Horizontal swipe through product grid on mobile
43. **Thumb-friendly quick add** - Large touch targets for adding to cart
44. **Collapsible vendor list** - Accordion view for mobile shop browsing

### COMPARISON TOOLS
45. **Product comparison** - Side-by-side up to 3 products
46. **Compare THC/CBD** - Visual bar charts for potency
47. **Compare pricing** - Price per gram analysis
48. **Compare terpenes** - If available in data

### TRUST & COMPLIANCE
49. **Verified grower badge** - Checkmark for verified partners
50. **Lab results preview** - THC/CBD on product cards link to full COA
51. **License numbers** - Visible on vendor shop pages
52. **Metrc integration indicator** - Shows state tracking compliance

### PERSONALIZATION
53. **Saved filters** - Remember preferred filter combinations
54. **Favorite strains** - Heart/star products for quick access
55. **Order history recommendations** - "Based on your orders"
56. **Price drop alerts** - Notify when favorites go on sale
